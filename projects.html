Weather App

In my free time I am working on a weather app using React. The app's state quickly became a lot to manage and I implemented Redux to help me manage it. Currently it is using the built in navigator object in browsers to locate the user sending the latitude and longitude of the user to Google's geocode API so the App with display their location and then to DarkSky's weather API to get their weather information and displays it in a component.
